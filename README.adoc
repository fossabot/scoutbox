= scoutbox

=== How to run?
. Clone project
[source, bash]
----
git clone https://github.com/feleko/scoutbox.git
cd scoutbox
----

. Install dependencies
[source, bash]
yarn install


. Download `google-services.json` from firebase console. Place this file in the `android/app/` directory.

. Run
[source, bash]
yarn run start
